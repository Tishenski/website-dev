ИНСТРУКЦИЯ: Как пересоздать базу данных для TeaPro
═══════════════════════════════════════════════════════════════

ПРОБЛЕМА: После переименования с кофе на чай, база данных 
не была пересоздана. Старая база называется coffeepro_db, 
а новая должна называться teapro_db.

═══════════════════════════════════════════════════════════════

СПОСОБ 1: Через phpMyAdmin (РЕКОМЕНДУЕТСЯ)
═══════════════════════════════════════════════════════════════

1. Откройте phpMyAdmin в браузере:
   http://localhost/phpmyadmin

2. В левом меню найдите базу данных "coffeepro_db" (если она есть)

3. Создайте новую базу данных:
   - Нажмите "Создать" (Create) в левом меню
   - Имя базы: teapro_db
   - Кодировка: utf8mb4_unicode_ci
   - Нажмите "Создать"

4. Выберите созданную базу "teapro_db" в левом меню

5. Откройте вкладку "Импорт" (Import)

6. Нажмите "Выбрать файл" и выберите файл:
   database.sql
   (находится в корне проекта)

7. Нажмите "Вперёд" (Go) внизу страницы

8. Дождитесь сообщения об успешном импорте

═══════════════════════════════════════════════════════════════

СПОСОБ 2: Через командную строку MySQL
═══════════════════════════════════════════════════════════════

1. Откройте командную строку (CMD) или PowerShell

2. Перейдите в папку проекта:
   cd C:\xampp\htdocs\website-coffee-dev

3. Выполните команду:
   C:\xampp\mysql\bin\mysql.exe -u root -e "source database.sql"

   ИЛИ если MySQL в PATH:
   mysql -u root < database.sql

═══════════════════════════════════════════════════════════════

СПОСОБ 3: Использовать готовый скрипт миграции
═══════════════════════════════════════════════════════════════

1. Откройте phpMyAdmin

2. Выберите базу данных "teapro_db" (или создайте её)

3. Откройте вкладку "SQL"

4. Скопируйте содержимое файла migrate_to_tea.sql

5. Вставьте в поле SQL запроса

6. Нажмите "Вперёд" (Go)

═══════════════════════════════════════════════════════════════

ПРОВЕРКА: Убедитесь что всё работает
═══════════════════════════════════════════════════════════════

1. В phpMyAdmin выберите базу "teapro_db"

2. Проверьте таблицы:
   - categories (должно быть 4 категории)
   - products (должно быть 4 товара)
   - users (должен быть 1 админ)

3. Обновите страницу сайта (F5)

4. Товары должны появиться!
